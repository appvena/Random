BAGAIMANA MENENTUKAN PILIHAN?
════════════════════════════════════════════════════════════════

Tanya diri sendiri:

1. Siapa yang pakai aplikasi ini?
   → Hanya saya sendiri? → PILIHAN 1
   → Tim 2-5 orang? → PILIHAN 2
   → Banyak orang (>5)? → PILIHAN 3

2. Dari mana buka aplikasi?
   → Dari file lokal (laptop/USB)? → PILIHAN 1 atau 4
   → Dari website online? → PILIHAN 3

3. Ada tech support/developer?
   → Tidak ada → PILIHAN 1 (skip tech files)
   → Ada → PILIHAN 2 atau 3 (include RELEASE-NOTES)

4. Perlu backup?
   → Tidak perlu (hanya test) → PILIHAN 1
   → Perlu (production) → PILIHAN 2, 3, atau 4

TROUBLESHOOTING: File mana yang perlu?
════════════════════════════════════════════════════════════════

"Aplikasi tidak bisa dibuka"
  → Pastikan random-workshop-v2.0.html ada
  → Buka dengan browser (bukan text editor)
  → Cek README.txt → Troubleshooting section

"Lupa cara pakai feature X"
  → Buka PANDUAN-v2.0.md, cari fitur X
  → Kalau tidak ada, buka README.txt

"Ada bug di aplikasi"
  → Cek RELEASE-NOTES-v2.0.txt → Known Issues
  → Jika tidak ada di situ, report ke developer

"Data tidak tersimpan"
  → Baca PANDUAN-v2.0.md → Troubleshooting
  → Cek localStorage di browser (F12)
